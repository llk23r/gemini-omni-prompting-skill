# Source boundary

## What is grounded in the Gemini Omni prompt guide

- Detailed prompting for output control
- Iterative editing through conversation
- Focused edits that preserve unchanged elements
- Camera angle and motion control
- Action and motion changes
- World knowledge for realism
- Text rendering with placement, animation, and exposure
- Complex action understanding without frame-by-frame input
- Reference fusion across image, video, text, and audio
- Style transfer with optional continuity preservation
- Storyboard-conditioned generation
- Consistency maintenance via references

## What comes from supporting sources

- General prompt structure and iteration patterns: Google Cloud prompt engineering guide
- Optional image-centric composition language and variation patterns: Gemini Image prompt guide

## Authoring rule

When extending this skill, add capabilities under an explicit `Extensions` heading rather than folding them into the core. Keep source attribution auditable.
